preview — poor man's preview statements for PR-s

This form generates markdown snippet that can be added to the PR comment manually to provide a preview of the new version as well as a diff file. It is poor man's version of the more 'official' PR-preview facility, in case the latter does not work...

(Note the the current version works only with ReSpec and plain HTML document. I am not sure how a preview version of Bikeshed should be generated.)
